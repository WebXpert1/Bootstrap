What is Programing?

Programing refers to the practice of writing code using languages like HTML, CSS, and JavaScript to create
functional and interactive websites, essentially building the structure, design, and dynamic behavior of a web
page through coding. .

Fields of Programing:

  1. Web Development
  2. Mobile Application Development
  3. Full stack Developer
  4. Front end Developer
  5. Backend systems
  6. Video game developer

Inline Elements 

Inline elements are elements that appear in a line,
rather than starting on a new line.They take up as 
much space as they need and don't break the flow of 
content.

Blockline Elements

block elements are building blocks of web pages
that start on a new line and take up the full 
width of the page. They are used to structure 
the main content of a webpage.

HTML Tags

HTML tags are the keywords on a web page that define how your web browser
must format and display your web page. Almost all tags contain two parts,
an opening and a closing tag.

  • Head
  • Image
  • Doctype
